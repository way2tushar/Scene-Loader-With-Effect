## Documentation

Drag and drop SceneLoaderCanvas prefab to your Hierarchy

#### Add this namespace

```C#
using way2tushar.Utility;
```

#### Use this function for loading your scene

```C#
StartCoroutine(SceneLoader.Instance.LoadScene(sceneName));
```
